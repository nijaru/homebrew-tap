# nijaru/homebrew-tap

Personal Homebrew tap for my tools.

## Install

```bash
brew tap nijaru/tap
```

## Tools

| Formula | Description                                                      |
| ------- | ---------------------------------------------------------------- |
| `og`    | Semantic code search — multi-vector embeddings + BM25            |
| `jb`    | Background job manager for AI agents                             |
| `bru`   | Fast, parallel package manager compatible with Homebrew formulae |
| `sy`    | Modern rsync alternative — fast, parallel file synchronization   |

```bash
brew install nijaru/tap/<formula>
```
